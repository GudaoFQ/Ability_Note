## Win 11配置成Win 10显示全部右键菜单

[toc]

### 方法一：临时显示完整右键菜单

在需要操作的区域按下`Shift`键的同时点击鼠标右键，此时弹出的就是完整的右键菜单。这种方式是临时的，松开`Shift`键后，再次右键点击仍会显示简化菜单。

### 方法二：通过命令提示符永久显示完整右键菜单（推荐）

1.  **打开命令提示符（管理员权限）**：
    - 按下`Win + S`组合键，打开搜索框。
    - 在搜索框中输入“命令提示符”，在搜索结果中找到“命令提示符”，然后选择“以管理员身份运行”。
    - 如果系统弹出用户账户控制提示框，请点击“是”以继续。
2.  **输入命令**：  
    在命令提示符窗口中，输入以下命令并按下`Enter`键：

```
reg.exe add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve
```

该命令通过修改[注册表](https://so.csdn.net/so/search?q=%E6%B3%A8%E5%86%8C%E8%A1%A8&spm=1001.2101.3001.7020)，将系统默认的右键菜单切换为Windows 10风格的完整菜单。  
3\. **重启资源管理器**：  
输入以下命令并按下`Enter`键，以重启资源管理器使设置生效：

```
taskkill /f /im explorer.exe & start explorer.exe
```

### 方法三：通过注册表编辑器永久显示完整右键菜单

1.  **打开注册表编辑器**：  
    按下`Win + R`组合键，打开“运行”对话框，输入`regedit`并按下`Enter`键。如果系统弹出用户账户控制提示框，请点击“是”以继续。
2.  **定位到指定注册表项**：  
    在注册表编辑器中，导航到以下路径：

```
HKEY_CURRENT_USER\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32
```

如果`InprocServer32`项不存在，可以右键点击`{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}`，选择“新建” -> “项”，并将其命名为`InprocServer32`。  
3\. **修改注册表值**：  
在右侧窗口中，确保默认值为空。如果默认值有内容，右键点击默认值，选择“修改”，将数值数据清空，然后点击“确定”。  
4\. **重启资源管理器**：  
按下`Ctrl + Shift + Esc`组合键打开任务管理器，在“进程”选项卡中找到“Windows 资源管理器”，右键点击它，然后选择“重新启动”。

### 恢复默认简化菜单

如果你想恢复到默认的简化右键菜单，可以在命令提示符（管理员权限）中输入以下命令并按`Enter`键：

```
reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f
```

然后再次输入以下命令重启资源管理器：

```
taskkill /f /im explorer.exe & start explorer.exe
```
