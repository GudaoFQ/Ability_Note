## C语言基础说明

### HelloWorld
```c
#include<stdio.h>
int main()//这个我的第一个 c 语言程序
{
    printf(“hello world\n”); //printf 是输出打印的函数
    return 0;
}
```
* `#include<stdio.h>` 头文件包含， 一定要有
* 每一个 c 语言的程序有且只有一个 main 函数， 这是整个程序的开始位置
* C 语言中()、 []、 {}、 “” 、 ’’、 都必须成对出现,必须是英文符号
* C 语言中语句要以分号结束。
* // 单行注释
* /* 信息 */ 多行注释
